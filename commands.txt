Anthony — Full Command Reference
=================================

A voice-controlled AI assistant for the GNOME desktop.
Say any of these naturally; exact phrasing is not required.


LAUNCHING & SEARCH
------------------
  "open firefox"
  "launch calculator"
  "start terminal"
  "open document.pdf"
  "open ~/Documents/report.txt"
  "go to amazon.com"
  "find screenshot.png"
  "search for reports"


WINDOW MANAGEMENT
-----------------
  "list windows" / "what applications are running?"
  "focus firefox" / "switch to text editor"
  "close terminal" / "quit firefox"
  "minimize window" / "maximize window"
  "restore window"
  "take a screenshot" / "screenshot firefox"
  "screenshot area"
  "move firefox to the right half"
  "tile left" / "snap to right"
  "move window to top-left corner"


KEYBOARD & MOUSE INPUT
-----------------------
  "type hello world"
  "press ctrl+c" / "press ctrl+v" / "press alt+tab"
  "press escape" / "press enter"
  "click at 500 300"
  "double click at 400 400"
  "right click at 600 500"
  "scroll down" / "scroll up"
  "drag from 100 100 to 500 500"


APP SHORTCUTS (context-aware)
-----------------------------
  "next tab" / "previous tab"
  "new tab" / "close tab"
  "copy" / "paste" / "cut"
  "select all"
  "undo" / "redo"
  "find" / "find in page"
  "zoom in" / "zoom out"
  "full screen"
  "new window"
  These use the correct shortcut for the focused app
  (e.g. Ctrl+T in Firefox, Ctrl+Shift+T in Terminal).

----------------

CONVERSATION MODE
-----------------
  "switch to chat mode" / "switch to conversation mode"
  "switch to command mode"
  "clear history" / "new topic"
  Ask any general question — Anthony will answer using its LLM.
  Intent is detected automatically: commands run tools,
  questions get conversational answers.
